Материалы для маркетологов. SemanticLD+JSON -- это вставки
семантической разметки на сайт для улучшения обработки информации
чат-ботами (GEO, Generative Engine Optimization). Исходные данные
брались с сайта. Пока в работе, не для выкладки.

Promotion содержит набор сгенерированных презентаций из тех же
промптов, которые использовались для семантической разметки. Приложены
как демонстрация возможностей LLM. Если вызовет интерес, можно
наладить производство подобных презентаций в промышленных
масштабах. Модели давалось задание представить компанию в максимально
свежих традициях подачи. Внутри есть свой README.txt с более
детальными пояснениями.
